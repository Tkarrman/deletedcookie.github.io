Milestone 1
Self-Publishing Publisher Partner

Vertical Slice

Deliver a polished first playable experience that demonstrates the core gameplay loop, tone, co-op chaos, and treatment systems.

Goal

Deliver a polished first playable experience that demonstrates the core gameplay loop, tone, co-op chaos, and treatment systems.

Focus

Core gameplay systems, player feel, first village, hub, treatment gameplay, and early community building.

Key Deliverables
  • Playable core loop
  • Initial village and hub
  • First illnesses and treatments
  • Basic crafting and resources
  • First enemies and village threats
  • Private playtest build
  • Early gameplay capture
Validation

Players understand the loop, enjoy the movement and chaos, and the build is stable enough for repeated multiplayer tests.

Outcome

A strong showcase build for publishers, partners, creators, and early community testing.

Milestone 4
Self-Publishing Publisher Decision Point

Early Access Release / Publisher Path Decision

If self-publishing, this is the Early Access release milestone. If a publisher path is secured, this becomes the decision point where production shifts toward the larger 1.0 plan instead.

Goal

Self-publishing path: release a stable Early Access version. Publisher path: use this milestone as the transition point into expanded full-production instead of launching Early Access.

Focus

Stability, optimization, onboarding, balancing, launch marketing, and release preparation.

Key Deliverables
  • Early Access build
  • Stability and optimization pass
  • Onboarding polish
  • Balancing pass
  • Marketing push
  • Launch materials
Validation

The game is stable, understandable, and fun enough for public Early Access players.

Outcome

Either a public Early Access launch for the self-publishing path, or a clear production handoff into the publisher-funded 1.0 roadmap.
